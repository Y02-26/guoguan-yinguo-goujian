# Counterfactuals And Mechanisms

Use this reference for the chapter's methods of constructing causal explanations.

## Why Counterfactuals Matter

The counterfactual asks:

```text
If the cause had not occurred, would the outcome still have occurred?
```

The difficulty is that researchers cannot directly observe both the actual outcome and the counterfactual outcome for the same case at the same time.

## Constructing A Reasonable Counterfactual

A useful counterfactual should:

- change the cause or key condition as little as possible;
- keep other relevant background conditions comparable;
- be historically and theoretically plausible;
- allow the researcher to infer how the outcome would likely change;
- generate statements that can be checked against empirical facts where possible.

## Counterfactual In Case Research

Case research can construct counterfactuals through:

- within-case comparison;
- comparison with similar cases;
- thought experiments;
- carefully chosen critical events or small contingent events;
- asking whether the actual case and counterfactual case remain comparable.

Warning: if changing X also changes many other background conditions, the counterfactual becomes weak.

## Causal Mechanism

The chapter treats causal mechanism as another major path for making causal inference in case research.

A causal mechanism is not just a variable and not just a chronology. It is a process in which entities and their activities connect cause and outcome.

Mechanism form:

```text
Cause X
  -> entity/activity 1
  -> entity/activity 2
  -> entity/activity 3
  -> outcome Y
```

## Elements Of A Mechanism

When building a mechanism, specify:

- the context in which the mechanism operates;
- the entities involved;
- the activities of those entities;
- the connection logic between activities;
- how one step generates or enables the next;
- how the chain produces or suppresses the outcome.

## Mechanism Is Not Mere Sequence

A chronological narrative says:

```text
A happened, then B happened, then C happened.
```

A mechanism explanation says:

```text
A changed the incentives/beliefs/capabilities of an actor,
which led to B,
which then generated C.
```

## Counterfactual vs Mechanism

| Path | Main question | Best use |
|---|---|---|
| Counterfactual | Would Y have differed without X? | judging whether X mattered |
| Mechanism | How did X produce Y? | explaining the production process |

The strongest causal explanation often uses both, but the chapter allows either as a key way to strengthen causal inference beyond temporal order and association.
