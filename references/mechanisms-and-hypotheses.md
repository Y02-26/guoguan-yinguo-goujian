# Mechanisms And Hypotheses

Use this reference to turn a causal idea into mechanisms, hypotheses, propositions, and observable implications.

## Mechanism Building

Start from the relation:

```text
X -> Y
```

Then force the missing middle:

```text
X -> actor change -> behavior change -> interaction/institutional change -> Y
```

## Mechanism Quality Criteria

A good mechanism should:

- Identify actors or entities.
- Specify what changes: beliefs, incentives, information, capabilities, legitimacy, constraints, coordination, or identity.
- Show temporal sequence.
- Explain why each step follows.
- Generate observable traces.
- Include conditions under which the mechanism is activated or blocked.
- Distinguish itself from rival mechanisms.

## Common IR Mechanism Types

### Coercion Mechanism

```text
Capability/threat X raises expected cost for target A.
A updates beliefs about punishment or resolve.
A changes policy or behavior.
Y occurs.
```

Observable traces: threats, military deployments, private warnings, elite deliberation about costs, policy concessions.

### Bargaining Mechanism

```text
Information/institution X changes bargaining range or credibility.
Actors revise expectations.
Agreement, deadlock, or escalation follows.
Y occurs.
```

Observable traces: negotiation records, offer changes, public/private signals, concession timing.

### Socialization / Norm Mechanism

```text
Normative environment X changes legitimacy standards.
Actors seek recognition, avoid stigma, or internalize rules.
Behavior shifts toward compliance or justified defiance.
Y occurs.
```

Observable traces: legitimacy language, legal justification, norm entrepreneurs, naming and shaming, institutional scripts.

### Domestic Coalition Mechanism

```text
External or institutional condition X reshapes domestic interests.
Coalitions form or fragment.
Leaders face new constraints or opportunities.
Policy Y occurs.
```

Observable traces: party debates, interest-group pressure, legislative vetoes, cabinet divisions, lobbying records.

### Threat Perception Mechanism

```text
Material or ideational signal X alters perceived threat.
Elites update beliefs about intentions or vulnerability.
Balancing, alignment, restraint, or escalation follows.
Y occurs.
```

Observable traces: threat assessments, speeches, intelligence estimates, alliance discussions.

### Learning Mechanism

```text
Experience X produces feedback.
Actors infer lessons.
Strategies, doctrines, or institutions change.
Y occurs.
```

Observable traces: after-action reviews, doctrinal change, memoirs, policy learning language, institutional reform.

### Institutional Constraint Mechanism

```text
Institutional rule X changes available choices or expected sanctions.
Actors adapt behavior to rules, monitoring, veto points, or enforcement.
Y occurs.
```

Observable traces: rule invocation, compliance reports, veto use, dispute-settlement records, monitoring documents.

## Hypothesis Forms

### Main Causal Hypothesis

```text
H1: Under condition S, the greater/presence of X, the more/less likely Y, because mechanism M.
```

Example structure:

```text
H1: In asymmetric alliance relationships, the higher a small state's security dependence on a patron, the more likely it is to align with the patron's crisis policy, because dependence increases the expected cost of defection and reduces credible outside options.
```

### Mechanism Hypothesis

```text
H1a: If mechanism M is operating, we should observe O1, O2, and O3 in the sequence X -> M -> Y.
```

### Scope Hypothesis

```text
H2: X's effect on Y is stronger when S is present and weaker or absent when S is absent.
```

### Rival Hypothesis

```text
HR: Y is better explained by R than by X; if HR is true, evidence OR should appear instead of OM.
```

### Negative / Null Expectation

```text
H0: Once rival cause R and confounder C are accounted for, X should have no independent effect on Y.
```

## Observable Implications

Observable implications translate mechanisms into evidence.

Ask:

- What should actors say if this mechanism is operating?
- What should they do first, second, third?
- What documents or events should appear?
- What timing pattern should be visible?
- What should not appear if the mechanism is false?
- How would rival mechanisms leave different traces?

Table:

| Mechanism step | Expected trace | Source type | Rival pattern |
|---|---|---|---|
| X changes beliefs | Elite statements, internal memos | archives, speeches, interviews | no belief update or different belief |
| Belief change alters behavior | policy shift, mobilization, concession | event data, decisions | behavior changes before belief shift |
| Behavior produces Y | outcome timing follows behavior | historical sequence | Y occurs without behavior |

## Deduction, Induction, And Abduction

### Deduction

Start from existing theory, derive hypotheses, then test. Use when theory is available and the research seeks to evaluate it.

Pattern:

```text
Theory -> assumption -> causal mechanism -> hypothesis -> evidence
```

### Induction

Start from empirical patterns, classify similarities and differences, then generate a tentative explanation. Use when the puzzle is new or under-theorized.

Pattern:

```text
Observed pattern -> recurring association -> tentative cause -> mechanism -> hypothesis
```

### Abduction / Retroduction

Start from a surprising fact, ask what hidden mechanism could make it intelligible, then test whether that mechanism leaves traces. Use when existing explanations fail.

Pattern:

```text
Anomaly -> possible hidden mechanism -> observable traces -> revised explanation
```

## From Topic To Hypothesis

Use this sequence:

1. Topic: “大国竞争与地区组织行为”
2. Puzzle: similar regional organizations respond differently to great-power pressure.
3. Y: degree of regional-organization alignment with a great power.
4. X: institutional dependence on great-power resources.
5. M: resource dependence changes elite incentives and agenda-setting power.
6. S: effect is stronger when alternative funding channels are weak.
7. H1: organizations with higher resource dependence are more likely to align their agenda with the patron, especially when alternative funding is scarce.
8. O: budget debates, agenda shifts after funding pressure, elite references to resource constraints.

## Hypothesis Checklist

- Does it name X and Y?
- Does it specify direction?
- Does it name the unit and scope?
- Does it include a mechanism or produce separate mechanism hypotheses?
- Can evidence support or weaken it?
- Is it distinguishable from rival explanations?
- Is it too broad for the available cases?
